02

プロジェクト概要 🎯

🎮基本情報

  • ジャンル: リアルタイム戦略ゲーム
  • プレイ人数: 5人対戦（1人 + CPU4人）
  • 開発環境: Unity 6.1 + C#
  • 開発手法: AI駆動開発（Cursor使用）

💡ゲームコンセプト

花を植えて領地を拡張し、最も多くの領地を獲得したプレイヤーが勝利する独創的な戦略ゲーム

03

技術スタック 🛠️

C#
Cursor (AI)
Stable Diffusion
